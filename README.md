# 📈 Student Performance Analysis

## 📌 Project Overview

Student Performance Analysis is a data analytics project that transforms raw academic data into meaningful and actionable insights. By analyzing students’ scores, attendance, and subject-wise performance, this project uncovers hidden patterns that influence academic success.

## 🎯 Objectives

* Analyze student academic performance
* Identify key factors affecting results
* Discover patterns in attendance and scores
* Provide data-driven insights for improvement

## 📊 Key Insights

* Relationship between attendance and performance
* Subject-wise strengths and weaknesses
* Impact of study habits on results
* Performance trends across students
